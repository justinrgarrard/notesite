Description

Notesite is a simple tool for displaying Markdown notes stored in their own Git repository.

Reading Notes

Go to the notes page to navigate between topics and entries.

Writing Notes

Notes are stored in their own Git repository and brought in via a Git Submodule. Whenever you update your notes repo, run the below commands to update this site.

$ git submodule update --remote
$ git add --all
$ git commit -m "Imported latest version of notes"
$ git push
